# SCT_DA_3
📊 HR Attrition Dashboard (Tableau)

📌 Overview
This project analyzes employee attrition using Tableau. The dashboard provides insights into employee distribution, attrition patterns, and key factors affecting employee turnover.


🎯 Objective
To identify patterns and trends in employee attrition and provide actionable insights for improving employee retention.


🛠 Tools Used
- Tableau
- Data Visualization
- HR Analytics Dataset


📊 Key Features
- KPI Cards (Total Employees, Attrition Count, Attrition Rate)
- Attrition by Department
- Attrition by Gender
- Attrition by Age (Binned Analysis)
- Interactive Dashboard Layout


🧠 Key Insights
- Overall attrition rate is ~16%, indicating moderate employee turnover  
- Sales department shows higher attrition compared to HR and R&D  
- Majority of employees are in the 25–40 age group  
- Younger employees (20–30) have higher attrition trends  
- Male employees show slightly higher attrition than female employees

🚀 Conclusion
The dashboard helps identify key attrition trends and supports data-driven decision-making to improve employee retention strategies.
